# GlazeWM Configuration

Personal configuration for [GlazeWM](https://github.com/glzr-io/glazewm), a tiling window manager for Windows.

## Features

- **Terminal**: Windows Terminal (`Alt+Enter`)
- **Browser**: Google Chrome (`Alt+B`)
- **Workspace switching**: `Alt+1` through `Alt+0`
- **Window management**: Move, resize, and tile windows using keyboard shortcuts
- **Focus navigation**: Vim-style keybindings (`Alt+HJKL`)

## Keybindings

| Action | Binding |
|--------|---------|
| Launch Terminal | `Alt+Enter` |
| Launch Browser | `Alt+B` |
| Focus window (left) | `Alt+H` |
| Focus window (right) | `Alt+L` |
| Focus window (up) | `Alt+K` |
| Focus window (down) | `Alt+J` |
| Move window | `Alt+Shift+HJKL` |
| Toggle floating | `Alt+Shift+Space` |
| Toggle fullscreen | `Alt+F` |
| Resize mode | `Alt+R` |
| Close window | `Alt+Q` |
| Pause bindings | `Alt+Shift+P` |

## Requirements

- [GlazeWM](https://github.com/glzr-io/glazewm) installed
- Windows 10/11
- Windows Terminal (for terminal shortcut)
- Google Chrome (for browser shortcut)

## Installation

1. Install GlazeWM
2. Copy `config.yaml` to `%APPDATA%\glazewm\config.yaml`
3. Restart GlazeWM
